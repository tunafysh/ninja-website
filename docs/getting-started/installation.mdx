## Installation Options

### 1. Prebuilt Binary

Download the latest release for your platform from the [GitHub releases page](https://github.com/tunafysh/ninja/releases):

```bash
# Extract and move to PATH
tar -xzf shurikenctl-*.tar.gz
sudo mv shurikenctl /usr/local/bin/
```

### 2. Package Manager

Use your platform's package manager where available (check releases for platform-specific packages).

### 3. Build from Source

Requires [Rust](https://rustup.rs/) toolchain:

```bash
git clone https://github.com/tunafysh/ninja.git
cd ninja
cargo build --release
sudo cp target/release/shurikenctl /usr/local/bin/
```

## Verify Installation

```bash
shurikenctl --version
shurikenctl help
```
