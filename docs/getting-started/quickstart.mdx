Get started with Ninja in 4 steps.

## 1. Install Ninja

Choose your preferred method:

```bash
# Option A: Download prebuilt binary
# Visit https://github.com/tunafysh/ninja/releases

# Option B: Build from source
git clone https://github.com/tunafysh/ninja.git
cd ninja
cargo build --release
sudo cp target/release/shurikenctl /usr/local/bin/
```

## 2. Install a Shuriken

```bash
shurikenctl install path/to/service.shuriken
```

The shuriken will be extracted to `~/.ninja/shurikens/{name}/`.

## 3. Configure (Optional)

If the shuriken uses configuration templates:

```bash
# Edit options
nano ~/.ninja/shurikens/{name}/.ninja/options.toml

# Generate config
shurikenctl configure {name}
```

## 4. Start the Service

```bash
# Start the service
shurikenctl start {name}

# Check status
shurikenctl list
```
